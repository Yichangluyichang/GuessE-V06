## 🚀 下一步操作

### 用户需要做的：

1. **创建配置文件**
   - 方法1：运行 `setup.bat` 自动设置
   - 方法2：手动复制 `config.example.js` 为 `config.js`

2. **获取 Gemini API Key**
   - 访问 https://makersuite.google.com/app/apikey
   - 登录并创建 API Key
   - 填入 config.js

3. **设置密码**
   - 在 config.js 中设置管理员密码

4. **开始使用**
   - 双击 index.html 打开工具
   - 使用密码登录
   - 开始生成和管理皇帝数据

## ⚠️ 重要提醒

1. **永远不要上传 config.js 到 GitHub**
