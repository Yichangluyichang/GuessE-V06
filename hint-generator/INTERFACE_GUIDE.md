## 操作流程图

### 生成新皇帝流程
```
输入皇帝名称 → 选择难度 → 点击"AI生成"
    ↓
等待AI生成（10-30秒）
    ↓
查看生成结果 → 编辑调整（可选）
    ↓
点击"同步到云端" → 完成
```

### 添加提示词流程
```
选择皇帝 → 点击"加载" → 查看现有提示词
    ↓
选择难度 → 点击"AI生成提示词"
    ↓
等待AI生成（5-15秒）
    ↓
查看生成的提示词 → 编辑（可选）
    ↓
点击"添加到列表" → 完成
```

### 评估提示词流程
```
选择皇帝 → 点击"开始评估"
    ↓
等待AI评估（20-40秒）
    ↓
查看评估结果 → 点击"应用所有修正"（可选）
    ↓
点击"同步到云端" → 完成
```

## 快捷键

- **Enter** - 在密码输入框中按 Enter 可直接登录
- **F12** - 打开浏览器开发者工具（查看错误信息）
- **Ctrl+R** - 刷新页面

## 使用技巧

1. **批量生成**：可以连续生成多个皇帝，每次生成后同步到云端
2. **编辑优化**：AI 生成后可以手动编辑优化提示词
3. **定期评估**：建议定期评估已有提示词，确保质量
4. **备份数据**：重要数据建议在 Firebase 控制台导出备份
